# enterprise-rag-knowledge-assistant

**Built an enterprise-style Retrieval-Augmented Generation system capable of ingesting PDF/TXT documents, performing hybrid retrieval using FAISS semantic search and BM25 keyword search, reranking results with a cross-encoder model, and generating grounded answers using an LLM. Implemented a Streamlit interface supporting document upload, conversational querying, source attribution, and automated grounding verification to detect hallucinations.**
# Enterprise RAG Knowledge Assistant

This project implements a Retrieval-Augmented Generation (RAG) system
designed for enterprise knowledge discovery across large document
collections.

Core components include:

- Document ingestion
- Chunking and embeddings
- Vector similarity search
- Context-aware LLM responses


Status: Active development
